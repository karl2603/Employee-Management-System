Employee Management System
==========================

A simple Employee Management System built using Java, JDBC, and MySQL that performs basic CRUD operations. This project demonstrates database connectivity, SQL operations, and backend development concepts.

--------------------------------------------------

Features
--------

- Add new employee
- View all employees
- Update employee details
- Delete employee records

--------------------------------------------------

Tech Stack
----------

- Java (Core Java)
- JDBC (Java Database Connectivity)
- MySQL (Database)
